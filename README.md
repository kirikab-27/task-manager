# タスク管理アプリ

カンバンボードとTODOリストを組み合わせた直感的なタスク管理アプリケーション

## 機能

- 📋 カンバンボード（ドラッグ&ドロップ対応）
- ✅ TODOリスト表示
- 🏷️ タグ/カテゴリ分け
- 🔍 検索・フィルター
- ⌨️ キーボードショートカット（Cmd/Ctrl+K でコマンドパレット）
- 📅 期限管理
- ⏱️ 見積時間の設定

## 技術スタック

- **フレームワーク**: Next.js 15 (App Router)
- **言語**: TypeScript
- **データベース**: SQLite + Prisma
- **スタイリング**: Tailwind CSS
- **状態管理**: Zustand
- **ドラッグ&ドロップ**: @dnd-kit

## 使い方

### キーボードショートカット

- `Cmd/Ctrl + K`: コマンドパレットを開く
- `Tab`: カンバン/リスト表示を切り替え
- `N`: 新規タスク作成
- `/`: タスク検索

### ビューの切り替え

左サイドバーから「カンバン」または「リスト」を選択してビューを切り替えます。

### タスクの作成

- ヘッダーの「新規タスク」ボタンをクリック
- またはキーボードショートカット `N` を押す
- コマンドパレット（`Cmd/Ctrl + K`）から作成

### タスクの管理

#### カンバンボード
- タスクカードをドラッグ&ドロップで別のカラムに移動
- カードをクリックして詳細を編集

#### TODOリスト
- チェックボックスをクリックして完了/未完了を切り替え
- タスクにホバーして編集・削除アイコンを表示

## スクリプト

- `npm run dev`: 開発サーバー起動
- `npm run build`: 本番ビルド
- `npm run start`: 本番サーバー起動
- `npm run db:push`: データベーススキーマを同期
- `npm run db:studio`: Prisma Studio起動（データベースGUI）
